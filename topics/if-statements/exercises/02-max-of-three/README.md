# Max of Three

## Problem

Write a method that returns the largest of three integers.

## Examples

```
maxOfThree(1, 2, 3)    → 3
maxOfThree(3, 2, 1)    → 3
maxOfThree(5, 5, 5)    → 5
maxOfThree(-1, -2, -3) → -1
maxOfThree(10, 3, 7)   → 10
```

## Hint

There are several approaches. One way: assume the first number is the max, then compare against the second and third, updating your answer if you find something larger. Another way: use nested if-else statements to check all possibilities.
