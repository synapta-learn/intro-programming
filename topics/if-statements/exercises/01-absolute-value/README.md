# Absolute Value

## Problem

Write a method that returns the absolute value of an integer. The absolute value is the number's distance from zero — it's always non-negative.

**Do not use `Math.abs()`** — implement it yourself using an if statement.

## Examples

```
absoluteValue(5)   → 5
absoluteValue(-5)  → 5
absoluteValue(0)   → 0
absoluteValue(-42) → 42
```

## Hint

If the number is negative, negate it (multiply by -1 or use the unary minus). Otherwise, return it as-is.
