# Exercise: Declare Variables

## Goal

Create variables of each basic type and return them from methods.

Implement these five methods:

- `getAge()` — return the integer `25`
- `getPrice()` — return the double `9.99`
- `isStudent()` — return the boolean `true`
- `getInitial()` — return the char `'J'`
- `getName()` — return the String `"Java"`

## Instructions

1. Open `Solution.java`
2. In each method, declare a variable of the correct type, assign the required value, and return it
3. For example, `getAge()` should declare an `int`, assign `25`, and return it

## Example

```java
public static int getAge() {
    int age = 25;
    return age;
}
```

## Hints

- Match the return type of each method to the variable type
- `char` uses single quotes: `'J'`
- `String` uses double quotes: `"Java"`
- `boolean` values are `true` or `false` (no quotes)
