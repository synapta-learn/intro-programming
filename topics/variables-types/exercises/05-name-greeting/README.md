# Exercise: Name Greeting

## Goal

Create a method that takes a first name and last name and returns a greeting message.

## Method Signature

```java
public static String greet(String firstName, String lastName)
```

## Expected Behavior

```
greet("Ada", "Lovelace")  → "Hello, Ada Lovelace! Welcome!"
greet("Alan", "Turing")   → "Hello, Alan Turing! Welcome!"
greet("Grace", "Hopper")  → "Hello, Grace Hopper! Welcome!"
```

## Instructions

1. Open `Solution.java`
2. Implement `greet` by concatenating the parts together with `+`
3. The format is: `"Hello, "` + first name + `" "` + last name + `"! Welcome!"`

## Hints

- Don't forget the space between the first and last name
- Don't forget the comma after "Hello"
- Don't forget the exclamation mark before " Welcome!"
