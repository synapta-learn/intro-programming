# Exercise: For Loop Sum

Write a Java method `sum(int n)` that returns the sum of all integers from 1 to n (inclusive).

## Examples

- `sum(5)` should return `15` (1+2+3+4+5)
- `sum(1)` should return `1`
- `sum(0)` should return `0`

## Instructions

1. Open `ForLoopSum.java` in the starter directory
2. Implement the `sum` method using a for-loop
3. Run the tests to verify your solution
