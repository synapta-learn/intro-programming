# Exercise: While Loop Countdown

Write a Java method `countdown(int n)` that returns a string counting down from n to 1, separated by spaces.

## Examples

- `countdown(5)` should return `"5 4 3 2 1"`
- `countdown(1)` should return `"1"`
- `countdown(0)` should return `""`

## Instructions

1. Open `WhileLoopCountdown.java`
2. Implement the `countdown` method using a while-loop
3. Run the tests to verify
