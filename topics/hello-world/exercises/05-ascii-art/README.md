# Exercise: ASCII Art

## Goal

Print this ASCII art box:

```
+--------+
|  JAVA  |
+--------+
```

## Instructions

1. Open `Solution.java`
2. Use `System.out.println()` to print each line of the box
3. The output must match exactly — count the dashes and spaces carefully

## Details

- The top and bottom borders are: `+` followed by 8 dashes `-` followed by `+`
- The middle line is: `|` followed by 2 spaces, then `JAVA`, then 2 spaces, then `|`

## Hints

- Count carefully: the box is 10 characters wide total
- Each line is its own `println` call
