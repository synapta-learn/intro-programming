# Exercise: Multi-Line Output

## Goal

Write a program that prints exactly these three lines:

```
Roses are red,
Violets are blue,
Java is fun,
And so are you.
```

Yes, that is actually four lines. Poems do not follow the rules.

## Instructions

1. Open `Solution.java`
2. Use one `System.out.println()` call for each line
3. The output must match exactly, including punctuation

## Hints

- You need four `println` statements, one per line
- Don't forget the commas at the end of the first three lines and the period at the end of the last line
