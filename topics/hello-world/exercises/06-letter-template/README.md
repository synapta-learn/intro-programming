# Exercise: Letter Template

## Goal

Print a formatted letter. Your output must match this exactly:

```
Dear Professor,

I am writing to confirm my enrollment
in the Introduction to Programming course.

I look forward to learning Java.

Sincerely,
Ada Lovelace
```

## Instructions

1. Open `Solution.java`
2. Use `println` calls to print each line
3. Use `System.out.println()` with no argument (empty parentheses) to print blank lines

## Important Details

- There is one blank line after "Dear Professor,"
- There is one blank line after "...course."
- There is one blank line after "...Java."
- The closing "Sincerely," and the name are on consecutive lines (no blank line between them)

## Hints

- `System.out.println()` with no text inside the parentheses prints just a blank line
- Count the blank lines carefully — getting them wrong is the most common mistake
