# Exercise: Print Your Name

## Goal

Write a program that prints a greeting with a name in this exact format:

```
My name is Ada Lovelace
```

## Instructions

1. Open `Solution.java`
2. Use `System.out.println()` to print `My name is Ada Lovelace`
3. You must use the name "Ada Lovelace" so the tests can verify your output

## Hints

- The entire sentence is a single `println` call
- Make sure there is exactly one space between each word
