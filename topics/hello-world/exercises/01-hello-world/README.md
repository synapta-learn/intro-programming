# Exercise: Hello, World!

## Goal

Write a program that prints exactly:

```
Hello, World!
```

This is the traditional first program in any language. Get it right, and you are officially a programmer.

## Instructions

1. Open `Solution.java`
2. Inside the `main` method, use `System.out.println()` to print `Hello, World!`
3. Make sure the output matches exactly — including the comma, space, and exclamation mark

## Hints

- The text must be inside double quotes
- Don't forget the semicolon at the end of the line
