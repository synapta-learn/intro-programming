# Exercise: Code Comments

## Goal

The starter code has several problems. Some lines that should be code are commented out, and some lines that should be comments are active code (causing errors). Fix the program so it compiles and produces this output:

```
Step 1: Start
Step 2: Process
Step 3: Done
```

## Instructions

1. Open `Solution.java`
2. Read through the code carefully
3. Uncomment lines that should be executable code
4. Comment out or remove lines that are not valid Java
5. The final program should print exactly the three lines above

## Hints

- A line starting with `//` is a comment — the compiler ignores it
- If a valid print statement is commented out, remove the `//` to make it active
- If a line is not valid Java and is not commented out, comment it out or delete it
- Look at each line and ask: "Is this valid Java? Should it run?"
