# Exercise: Basic Arithmetic

## Goal

Implement four methods that perform the basic arithmetic operations on two integers.

## Method Signatures

```java
public static int add(int a, int b)
public static int subtract(int a, int b)
public static int multiply(int a, int b)
public static int divide(int a, int b)
```

## Expected Behavior

```
add(10, 3)       → 13
subtract(10, 3)  → 7
multiply(10, 3)  → 30
divide(10, 3)    → 3    (integer division — no decimals)
```

## Instructions

1. Open `Solution.java`
2. Implement each method using the appropriate arithmetic operator
3. `divide` performs integer division (no decimal part)

## Hints

- `+` for addition, `-` for subtraction, `*` for multiplication, `/` for division
- Remember: dividing two ints gives an int result (decimal part is discarded)
