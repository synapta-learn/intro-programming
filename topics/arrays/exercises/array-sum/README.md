# Array Sum

## Problem

Write a method `sum(int[] numbers)` that returns the sum of all elements in the array.

## Examples

| Input | Output |
|-------|--------|
| `{1, 2, 3, 4, 5}` | `15` |
| `{10, -3, 7}` | `14` |
| `{42}` | `42` |
| `{}` | `0` |

## Hints

- Start with a sum variable set to `0`
- Loop through every element and add it to the sum
- An empty array has a sum of `0`
